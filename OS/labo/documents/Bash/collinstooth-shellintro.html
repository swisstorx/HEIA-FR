Introduction to Shells


Starting a Shell

  • Each user has a default shell, defined in a /etc/passwd file.
  • See what shell you are in:
    bash-2.05b$ echo $SHELL
/bin/bash
  • You can start another shell from the default shell by simply typing the name of the shell, then get back to where you were by typing exit: 
    bash-2.05b$ dash
$ exit
bash-2.05b$
  • Files you create can be run using a specified shell by creating a PATH to the shell at the top of the file, e.g.:
    (instead of 'pico', you can use your own favorite editor, e.g. emacs, vi etc.) 
    bash-2.05b$ pico my.sh   
#!/bin/csh
#apart from the first line ('shebang line'), comments are signified by hashes
#
#note: the setenv command does not exist in bash or zsh. This means
#the 'shebang line' has specified the csh interpreter
#
setenv dummyvariable "323"
echo "I just set dummyvariable to $dummyvariable"
  • The line #!/bin/csh is often called the "shebang" line.
  • There are a number of different shells. The main ones found on modern LINUX/UNIX systems are tcsh and bash
  • The ubuntu cluster has bash and dash but does not have tcsh or csh.
  • From now on in this course, bash and perl are used.
    • bash: provides easy access to LINUX commands.
    • perl: provides access to LINUX commands, has extra functionality and allows easy string manipulation. (It is also the language that many web scripts are written in.)

A Basic Shell Program

  • First, use an editor like (pico, emacs or vi) to create a file containing the text in the grey box:
    ex1.sh: My first shell script: 
    
#!/bin/bash
#
# C Collins-Tooth
# A basic script to print one line to the standard output.
#
echo "this is my first script - HELLO WORLD!"
  • Save the file as ex1.sh
  • The file does not *need* to have a specific file extension. (For the examples given in ths course, bash scripts are .sh and perl scripts are .pl.
  • Then, make the file executable
    • bash-2.05b$ chmod u+x ex1.sh
  • Finally, run the script: 
    bash-2.05b$ ./ex1.sh
this is my first script - HELLO WORLD!
bash-2.05b$
